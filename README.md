# DER - DIE - DAS

This is a (work in progress) project. The user could select the gender of a word in German and discover if it is right!

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
